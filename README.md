# 🗼 Japan_Sync

Un guide de voyage numérique et interactif conçu pour planifier un itinéraire complet de 25 jours au Japon (Tokyo, Kyoto, Osaka). L'application propose une interface moderne, fluide et responsive pour suivre le programme au jour le jour.

## 🚀 Fonctionnalités

*   **Itinéraire interactif :** Navigation jour par jour à travers les différentes étapes du voyage.
*   **Design responsive :** Parfaitement optimisé pour une utilisation sur smartphone pendant le voyage.
*   **Données en temps réel :** Intégration complète avec Firebase pour la gestion et la synchronisation des étapes.
*   **Cartographie intégrée :** Intégration avec les API de navigation pour repérer facilement les points d'intérêt.

---

## 🛠️ Technologies utilisées

*   **Front-End :** HTML5, CSS3, JavaScript (ES6+)
*   **Base de données & Backend :** Firebase (Firestore)
*   **Hébergement :** GitHub Pages
